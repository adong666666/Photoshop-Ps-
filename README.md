PhotoShop脚本(PhotoShop Script)是：

* Photoshop最神秘，也是最强大的一个功能。

* 它可以允许您以代码的方式来操作文档、美化图像、处理图层、控制通道、编辑选区、使用滤镜等等，就像在Photoshop界面上操作一样。

* PhotoShop脚本主要用于重复性的任务或用于制作非常复杂的特殊效果。
